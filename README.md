# CBT-AI-Emotional-Suppor-Agent
# 基于CBT的情绪支持 AI Agent

A project exploring an AI agent capable of emotion-aware conversation and interaction.  
This system is designed as a multi-stage conversational AI agent with structured workflow and prompt architecture.  
（本项目旨在设计一个能够感知用户情绪的 AI 对话智能体，包含分阶段对话流程与 prompt 架构设计）

---

## Project Overview
## 项目概述

This project focuses on designing an AI conversational agent that can recognize user emotions  
and respond appropriately through staged interactions.  
It is inspired by cognitive-behavioral principles for structured dialogue,  
but does not provide any medical or psychological counseling.  
（项目重点在于设计一个能够识别用户情绪并做出适当回应的 AI 对话智能体，互动对话的理论基础来自认知行为理论（CBT），但不提供任何医疗或心理咨询）

## Example Conversation
## 对话示例

**User:** I feel stressed about work today.  
（用户：我今天工作压力很大）

**AI Agent:** I hear you are feeling stressed.  
Can you tell me what part of your day was most challenging?  
（AI Agent：我听到你感到有压力。可以跟我说说今天最让你紧张的事情吗？）
